# 🔠 Text to Morse Converter
This Python application allows users to convert text into Morse code using a simple and intuitive graphical user interface (GUI) built with Tkinter. The app is designed to be user-friendly, enabling quick and easy text-to-Morse translation. With this tool, you can convert any message into Morse code and vice versa.


## Installation 🛠️
```python
#Clone this repository:
git clone https://github.com/harris8099/Text-to-Morse-converter-.git
#Install the required dependencies:
pip install tkinter
```
```python
#Run the application:
python morse_converter.py
```
